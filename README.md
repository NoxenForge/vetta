# i18n Starter

A minimal Next.js starter with multi-language support via next-intl.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:4000](http://localhost:4000).

## Supported Locales

- English (en) — default
- 简体中文 (zh)
- 繁體中文 (tw)

## Tech Stack

- Next.js 15
- React 19
- next-intl
- Tailwind CSS v4
